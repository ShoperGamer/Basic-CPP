### **Intelligent Address Input System in C++**

The `SmartAddressInput` project is a C++ console application that demonstrates user input handling, with a specific focus on managing default arguments for addresses. If the user does not provide an address, the program will automatically use a predefined default value ("Bangkok"). This application combines the use of `cin` and `getline` along with input buffer management for a flexible and user-friendly input experience.

----------

### ‚ú® **Key Features**

-   **üë• Name and Address Input:** Capable of receiving both name and address information from the user.
-   **üìç Default Address:** Automatically sets "Bangkok" as the default address if no input is provided by the user.
-   **üí° Intelligent Input Handling:** Utilizes `getline` to capture input including spaces, and `cin.ignore` to manage the input buffer, preventing common input issues when mixing `cin` and `getline`.
-   **‚öôÔ∏è `showdata()` Function:** Displays the user's name and address information in a clear and organized format.
